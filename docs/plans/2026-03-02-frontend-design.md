# 前端聊天界面设计文档

## 概述

为 ETL Agent 添加一个单 HTML 文件的聊天界面，通过 WebSocket 与后端交互，支持 Markdown 渲染和 Mermaid 图谱展示。

## 技术选型

- **单 HTML 文件**：内嵌 CSS + JS，零构建工具
- **CDN 依赖**：`marked.js`（Markdown 渲染）、`mermaid.js`（图谱渲染）、`highlight.js`（SQL 语法高亮）
- **挂载方式**：FastAPI `StaticFiles` + 根路由重定向

## 视觉风格（参考截图）

- 浅色主题，白色背景 `#f7f8fa`
- Agent 消息：左侧，紫色圆形图标 + 浅灰白色卡片，圆角阴影
- 用户消息：右侧，紫色圆角气泡 `#6c5ce7`，白色文字
- SQL 代码块：深色背景 `#1e1e2e` + 语法高亮（绿色关键字）
- 表格：清爽灰白交替行
- 内联代码：紫色/绿色背景标签
- 步骤进度：轻量灰色提示条

## 消息类型处理

| WebSocket `type` | UI 行为 |
|---|---|
| `response` | Agent 消息气泡，marked.js 渲染 Markdown，mermaid 渲染图谱 |
| `step_progress` | 居中灰色步骤提示条 |
| `done` | 恢复输入框可用 |
| `error` | 红色错误消息 |

## 改动文件

| 文件 | 操作 | 说明 |
|---|---|---|
| `static/index.html` | 新增 | 前端聊天界面（唯一文件） |
| `main.py` | 修改 | 挂载 static 目录 + 根路由 |
